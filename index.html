What We Built

Automatic Lift System for Industrial Precision

Industrial automation demands precision lifting solutions that are reliable, efficient, and maintenance-free. Our project delivers exactly that with a completely automatic approach.

Instead of relying on bulky hydraulic cylinders or manual operation, we engineered a system where two specially-shaped chains interlock to form a rigid pushing column. This zip chain technology creates a strong, self-locking mechanism perfect for industrial lifting applications.

  • Automated Operation
  • Compact Design
  • High Reliability
  • Precise Control

System Design

Control + Mechanics Unified

The architecture connects power electronics, sensors, PLC logic, and lift mechanics in one synchronized framework. This section uses your core design illustrations and implementation visuals to represent the complete end-to-end system.

  • PLC Controller: Executes lift, stop, and return logic with fast signal processing and electrical interlocks.
  • Zip Chain Drive: Converts rotational motor output into rigid linear lifting action for high-speed movement.
  • Sensing + Safety: Uses proximity sensing, overload protections, and emergency paths to secure operation.
  • Power Module: Delivers stable regulated DC power for control electronics and feedback circuitry.

System Architecture

Detailed View of Core Components

Move left to right and right to left to understand what is behind each core module and how the full lift system behaves as one machine.

Panthograph Lifter

Behind Panthograph Lifter

The pantograph lifter transforms linear actuator force into a stable scissor motion. The geometry keeps platform travel vertical, while distributing load evenly across pivot joints to reduce wobble at high speed.

  • Multi-link frame converts compact motion into tall extension
  • Pivot synchronization helps maintain lift balance
  • Mechanical design supports fast rise and controlled return

Zip Chain Assembly

Behind Zip Chain Assembly

The zip chain system links and locks chain elements into a rigid column. When driven by the motor, it provides long linear travel in a compact footprint, replacing bulky hydraulic setups for this use case.

  • Rigid chain column forms only during extension
  • High travel speed with repeatable precision
  • Compact storage path when retracted

Delta PLC Controller

Behind Delta PLC Controller

The Delta PLC is the logic core that sequences start, stop, safety, and return states. It reads sensor inputs, applies ladder logic, and controls outputs in deterministic scan cycles for reliable automation.

  • Deterministic scan loop for predictable control
  • Interlocks prevent unsafe motor actions
  • Easily expandable I/O for future modules

PLC Interface

Behind PLC Interface

The interface layer maps field devices to PLC terminals, ensuring clean signal routing and clear diagnostics. Proper wiring segregation between power and logic reduces electrical noise and communication faults.

  • Structured terminal mapping improves maintainability
  • Signal conditioning stabilizes input readings
  • Status indicators speed up troubleshooting

Inductive Sensor

Behind Inductive Sensor

The inductive sensor detects metal targets without physical contact, providing reliable position feedback for stop points and safety checks. This contactless detection improves life in dusty industrial environments.

  • Contactless sensing reduces wear and maintenance
  • Fast switching supports dynamic control loops
  • Reliable trigger points improve operational safety
Parts We Used

Inside the Build

Delta PLC (DVP-14SS2)

This is the brain - takes inputs from sensors and decides what the motor should do.

  • Handles 8 input signals
  • Controls 6 output devices
  • We wrote the logic in ladder diagram

Proximity Sensor (Omron)

Detects when something is nearby without physical contact and triggers lift action.

  • Works in harsh conditions
  • Detects from a distance
  • Waterproof design

Power Supply Unit

Converts AC mains to clean 24V DC that powers the complete control circuit.

  • Stable 24 volt output
  • Heat-resilient operation
  • Built-in safety features

Zip Chain Actuator

The star mechanism where two chains interlock to create a rigid lifting column.

  • Pushes up and pulls down
  • Handles heavy loads
  • Folds compactly when idle
